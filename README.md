# ISKACAME-MK

> A scraping tool for demonstrational purposes only.

A demo website which effectively has a crawler that reads stories from `@iskacame` on Instagram
and then proceeds to put them in simple UI cards for users to look at. Made without any ill intention towards `@iskacame`,
it's just a way for the author to practice their ruby skills and pump their CV. If any administrators from `@iskacame` would like to get in touch about the content on this website, 
feel free to contact the author `@kralmarko`.
